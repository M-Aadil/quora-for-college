# College Quora 

An online student forum where an student can post his query and will get a relevant answer from another student or faculty (Specially from Jamia Millia Islamia)


## Technology Used

### Front End
- HTML
- CSS
- JavaScript
- ReactJs

### Backend
- NODE
- Express
- MySQL
